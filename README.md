# Nest.js Demo with Fastify and Swagger

教程文档链接： <https://leader.js.cool/#/basic/framework/nest>

## 安装

```bash
$ npm install
```

## 运行

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

生成自动文档：

```bash
npx compodoc -p tsconfig.json -s
```

## 测试

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
